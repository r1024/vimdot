# vimdot
极简vim配置

# 安装方法
```
git clone https://github.com/r1024/vimdot.git && cd vimdot && ./install_simple.sh
```

# 支持特性
```
" -> 0: vim-plug                    [插件管理]
" -> 1: nerdtree                    [目录树]
" -> 2: nerdcommenter               [注释工具]
" -> 3: vim-interestingwords        [高亮选中]
" -> 4: vim-airline                 [状态栏]
" -> 5: vim-choosewin               [窗口切换]
" -> 6: undotree                    [回退工具]
```
